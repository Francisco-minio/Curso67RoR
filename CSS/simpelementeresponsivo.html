@media queries

CSS uses the @media rule to detect information about the device beign used to access the website. For example, it can be used to detect the size of the device, its capabilities, and its orientation. For this course, we will be using it to detect the device size, but keep in mind that it is capable of much more.

Desktop width greater than or equal to 1024px

Tablet width beetwen 481 to 1023px

Phone width less than or equal to 480px
